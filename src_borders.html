borders

Dean Koch June 10, 2020

This is the initial setup script for reproducing the rasterbc dataset. It downloads some shapefiles to define BC borders, sets up config for preprocessing.

The geographical projection system is set to match the one used in the {bcmaps} package (Albers NAD83, standard in BC). Resolution and extent are set to match and extents are defined for each individual output tile (10km x 10km mapsheet).
